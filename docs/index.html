The World

A Dwarf Fortress-style colony simulation where AI agents create emergent stories.

View on GitHub

Quick Start

git clone https://github.com/antwisearch/the-world.git
cd the-world
python -m src.api

Then open http://localhost:8080

Features

  • AI Agents with needs and jobs
  • Procedural world with biomes
  • Emergent storytelling
  • Economy and trade

Built with Python + FastAPI
