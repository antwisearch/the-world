# The World

A Dwarf Fortress-style colony simulation where AI agents live, work, and create emergent stories.

## Quick Start

```bash
git clone https://github.com/antwisearch/the-world.git
cd the-world
source venv/bin/activate
python -m src.api
```

Open http://localhost:8080

## What is The World?

The World simulates a colony where:
- Agents have needs (food, water, shelter, happiness)
- Agents work jobs (gatherer, builder, hunter, farmer, trader, guard)
- The world generates biomes, resources, and events
- Stories emerge naturally from the simulation

## Screenshots

The game runs in your browser with ASCII graphics:
- Agents shown as letters (h=hunter, b=builder, f=farmer, etc.)
- Resources: T=tree, ~=water, *=food
- Buildings: [#]=shelter, +=farm
